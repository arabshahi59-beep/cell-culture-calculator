# Cell Culture Media Calculator

A desktop application for immunology and cell biology labs to quickly calculate media compositions.

## Features
- Calculate FBS, Pen/Strep, L-Glutamine, and supplements
- Support for multiple cytokines
- Save/Load recipes
- Presets for common cell types (T cells, B cells, Macrophages)
- Export results to text files
- Light/Dark/System appearance modes
